# AGENTS.md — graphene-lang

Read the organization-level `AGENTS.md` (one directory up, org root) first;
its rules apply here in full.

## Scope of this repository

Pkl core schemas, the codegen toolchain, the service IDL, version glue, and
the eval artifact format. Nothing here executes workflows — this repository
defines the typed model and its generated bindings.

## Rules

- Pkl schemas are the single source of truth. Generated code is never edited
  by hand and always regenerated by the toolchain from `bin/`.
- Schemas and generated bindings are released together as one versioned
  artifact; never publish one without the other.
- Any change to a published schema follows the schema evolution rules
  (to be defined by ADR in `graphene-docs/adr/`).
- Stack/technology choices are recorded as ADRs in `graphene-docs` before
  implementation starts.
